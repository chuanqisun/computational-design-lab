# Prompt Library

A single page prompt configurator app that helps user build a good prompt for product design related tasks, including:

- Product design visualization
- Conceptual ideation
- High-fidelity rendering
- Divergent and convergent brainstorming
- Synthetic user persona generation
- Synthetic user testing

## UI/UX

- Minimalist, high space efficiency, respect browser default style and intrinsically responsive
- Multi-functional: both a production tool, and a learning improve to enhance user's prompting literacy
- Support text to text, text to image, image to text, image to image by generalizable modular configuration

## Tech stack

- Vanilla js, html, css
- Gemini SDK for prompt rewrite, enhancement, playground
